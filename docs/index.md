# 🐍 Python DSA Notes

> *"Simple is better than complex. Complex is better than complicated."*
> — **Tim Peters** (The Zen of Python)

Welcome to **Python DSA Notes** — a structured, beginner-to-advanced reference for **Python Programming** and **Data Structures & Algorithms**, covering everything from Python basics to advanced algorithms and interview-level practice problems.

---

## 🗺️ What's Inside

| # | Section | What You'll Learn |
|---|---------|-------------------|
| 01 | 🔵 **Basics** | Variables, data types, loops, functions, error handling, and modules |
| 02 | 📊 **Complexity Analysis** | Big O, Big Θ, Big Ω, time and space complexity |
| 03 | ⚙️ **Algorithms** | Sorting, searching, recursion, dynamic programming, greedy, and graph algorithms |
| 04 | 🗃️ **Data Structures** | Lists, tuples, sets, dicts, stacks, queues, linked lists, trees, heaps, and graphs |
| 05 | 🏆 **Practice Problems** | Curated problems from easy to hard — interview ready |

---

## 🚀 How to Use These Notes

!!! tip "New to Python?"
    Start with **Basics → Complexity Analysis → Data Structures → Algorithms** before jumping into practice problems.

!!! info "Already know Python?"
    Jump straight to **Data Structures** or **Algorithms** depending on what you're revising.

!!! warning "Keep in mind"
    Every topic includes real code examples, memory diagrams, and practical use cases — read them carefully, don't just skim the code.

## 🛠️ Run Locally

```bash
# Clone the repo
git clone https://github.com/yxshkm-notes/python-dsa-notes.git
cd python-dsa-notes

# Create virtual environment
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows

# Install dependencies
pip install mkdocs mkdocs-material

# Serve locally
mkdocs serve
```

Open [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser.
