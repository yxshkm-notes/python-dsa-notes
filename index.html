    Welcome to Python DSA Notes 👋¶

    "Simple is better than complex. Complex is better than complicated." — Tim Peters (The Zen of Python)

    📖 About This Documentation¶

    This is a personal reference guide for Python Programming and Data Structures & Algorithms (DSA). It covers everything from the basics of Python syntax to advanced data structures and algorithms — written in a clean, beginner-friendly format with real examples and code snippets.

    Whether you're just starting out or revising for interviews, this doc has you covered. 🚀

    🗂️ What's Inside¶

    🔵 Basics¶

    Core Python language fundamentals every programmer must know.

    Topic Description
    Variables & Data Types int, float, str, bool, None and more
    Input & Output print, input, f-strings, format
    Control Flow if, elif, else, ternary expressions
    Loops for, while, break, continue, range
    Functions def, arguments, return, scope, lambda
    List Comprehensions Compact loops, filtering, nested comprehensions
    Error Handling try, except, finally, raising exceptions
    Modules & Imports import, from, as, standard library
